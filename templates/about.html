À Propos de Mon Projet

Bienvenue sur mon application web personnelle ! Ce projet a été réalisé en solo dans le cadre d'un exercice visant à exploiter une API pour enrichir le contenu de l'application.

J'ai choisi de travailler avec l'API [Nom de l'API] qui propose une architecture REST et retourne des données au format JSON. Pour garantir l'accessibilité, j'ai pris en compte la gestion du token lorsque nécessaire, que ce soit pour une API publique, partenaire, ou privée.

Ce projet utilise au moins trois endpoints différents de l'API sélectionnée. Ces endpoints ont été soigneusement choisis pour diversifier le contenu et offrir une expérience utilisateur riche et engageante.

Si vous avez des questions, des commentaires ou des suggestions pour améliorer ce projet, n'hésitez pas à me contacter. Merci de découvrir mon travail !
